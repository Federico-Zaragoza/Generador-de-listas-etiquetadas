Notificaciones

Demostración del Sistema de Notificaciones

Haz clic en los botones para generar diferentes tipos de notificaciones:

Características:

  • Notificaciones en tiempo real con animaciones
  • Contador de notificaciones no leídas
  • Diferentes tipos de notificaciones (info, éxito, advertencia, error)
  • Notificaciones toast que aparecen y desaparecen automáticamente
  • Marca notificaciones como leídas/no leídas
  • Elimina notificaciones individuales o todas juntas
  • Diseño responsive para móviles y escritorio

Cómo implementar:


// 1. Incluir CSS y JS en tu HTML
<link rel="stylesheet" href="src/css/style.css">
<script src="src/js/notifications.js"></script>

// 2. Agregar HTML básico
<div class="notification-container">
    <button class="notification-btn">
        <i class="fas fa-bell"></i>
        <span class="notification-badge">0</span>
    </button>
    <div class="notification-menu">
        <div class="notification-header">
            <h3>Notificaciones</h3>
            <div class="notification-actions">
                <button class="mark-all-read">Marcar todo como leído</button>
                <button class="clear-all">Limpiar todo</button>
            </div>
        </div>
        <div class="notification-list"></div>
    </div>
</div>

// 3. Crear notificaciones en tu código
window.notificationSystem.addNotification({
    type: 'success',  // 'info', 'success', 'warning', 'danger'
    title: 'Operación completada',
    message: 'La tarea se ha completado exitosamente.',
    time: new Date(),
    unread: true,
    link: '/optional-link.html'  // opcional
});
